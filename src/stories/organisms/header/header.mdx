import { Header } from './Header';

# Header Component

Header component for the application.

## Usage

```jsx
<Header
  user={{ name: 'John Doe' }}
  onLogin={() => console.log('Login clicked')}
  onLogout={() => console.log('Logout clicked')}
  onCreateAccount={() => console.log('Create Account clicked')}
/>

## Props

| Name             | Type                   | Default     | Description                           |
| ---------------- | ---------------------- | ----------- | ------------------------------------- |
| `user`           | `{ name: string }`     | `undefined` | User object containing user details. |
| `onLogin`        | `() => void`           | `undefined` | Click handler for the Login button.   |
| `onLogout`       | `() => void`           | `undefined` | Click handler for the Logout button.  |
| `onCreateAccount`| `() => void`           | `undefined` | Click handler for the Sign Up button. |
