# Button Component

Primary UI component for user interaction.

## Usage

```jsx
<Button label="Click me" onClick={() => console.log('Button clicked')} />

## Props

| Name             | Type                    | Default     | Description                                               |
| ---------------- | ----------------------- | ----------- | --------------------------------------------------------- |
| `primary`        | `boolean`               | `false`     | Specifies if this button is the principal call to action.|
| `size`           | `'sm' | 'md' | 'lg'`    | `'md'`      | Determines how large the button should be.                |
| `backgroundColor`| `string`                | `undefined` | Specifies the background color of the button.             |
| `label`          | `string`                | `''`        | The content displayed on the button.                      |
| `onClick`        | `() => void`            | `undefined` | An optional click handler function.                       |
